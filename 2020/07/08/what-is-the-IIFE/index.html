立即执行函数（IIFE）是什么

我们定义函数的方法是：

如果我们需要在定义函数之后立即调用，并不能直接在函数的定义后面加上()，这会产生错误。因为 function 关键字可以当作语句，也可以当作表达式。JavaScript 引擎规定如果 function 关键字出现在行首，则一律解释为语句。

所以当 JavaScript 引擎识别到 function 关键字在行首时，会认为这一段是函数的定义，不能在后面直接加圆括号调用，产生报错。

因此可以在 function 关键字前面添加符号，让引擎将其理解成一个表达式。而常用的做法是将其放在一个圆括号里面。

这就是立即调用的函数表达式，即立即执行函数（Immediately-Invoked Function Expression），简称 IIFE。

这种写法最后的分号都是必须的。如果省略分号，遇到连着两个 IIFE，JavaScript 会将它们连在一起解释，将第二行解释为第一行的参数，可能就会报错。

一般来说，只对匿名函数使用 IIFE，好处是：

  1. 不用为函数命名，避免污染全局变量
  2. IIFE 内部形成一个单独作用域，可以封装一些外部无法读取的私有变量

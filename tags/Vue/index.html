  • 用 Vue3 创建项目

    2020-08-02 |
    • { Vue }
    1. 用命令行定位到想创建项目到目录
    2. 全局安装 create-vite-app

      yarn global add create-vite-app@1.18.0
      或者
      npm i -g create-vite-app@1.18.0

    3. 创建项目目录

      create-vite-app 项目名称
      或者
      cva 项目名称

    4. 运行项目
      移动到项目目录下：cd 项目名称
      安装依赖：npm install（或者 yarn）
      运行：npm run dev（或者 yarn dev）
